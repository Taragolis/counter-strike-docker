# Docker image for Half-Life Dedicated Server

### Build an image:

```
 $ make build
```

### Build an image and start new Counter-Strike 1.6 server:

```
 $ make build run
```

### Start the server:

```
 $ make run
```

### Stop the server:

```
 Ctrl+C
```
